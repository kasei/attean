=head1 NAME

Attean::API::BlankOrIRI - Role representing blank or IRI terms

=head1 VERSION

This document describes Attean::API::BlankOrIRI version 0.013

=head1 SYNOPSIS

  use v5.14;
  use Attean;

=head1 DESCRIPTION

This is a Moo role that both Blank and IRI terms consume, and is used as a
constraint for triple and quad subjects.
