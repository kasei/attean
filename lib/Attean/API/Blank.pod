=head1 NAME

Attean::API::Blank - Role representing RDF blank terms

=head1 VERSION

This document describes Attean::API::Blank version 0.012

=head1 SYNOPSIS

  use v5.14;
  use Attean;

=head1 DESCRIPTION

This is a Moo role that Blank RDF terms consume.

=head1 ROLES

This role consumes the L<Attean::API::Term> and L<Attean::API::BlankOrIRI> roles.

=head1 METHODS

This role provides default implementations of the following methods:

=over 4

=item C<< ebv >>

Returns the boolean value of this term.

=item C<< compare( $blank ) >>

Returns -1, 0, or 1 if C<< $blank >> is less than, equal to, or greater than
the referent based on SPARQL sorting order.

=back
